## Marketplace

通常指 NFT 交易交易平台，主要包括：

1. OpenSea
- 最大的 NFT 交易平台
- 支持多链：Ethereum、Polygon、Solana 等
- 支持多种 NFT 标准：ERC-721、ERC-1155

2. Blur
- 专注于 NFT 交易的聚合平台
- 提供专业交易工具
- 零交易费用

3. X2Y2
- 去中心化 NFT 交易平台
- 支持批量挂单和购买
- 交易费用返还给平台用户

4. LooksRare
- 社区驱动的 NFT 市场
- 通过 LOOKS 代币激励用户
- 支持 NFT 质押

5. Rarible
- 去中心化 NFT 创作和交易平台
- 多链支持
- 通过 RARI 代币进行治理

这些平台各有特色，用户可以根据自己的需求选择合适的交易平台。主要考虑因素包括：
- 交易费用
- 流动性
- 安全性
- 用户界面
- 支持的区块链网络
